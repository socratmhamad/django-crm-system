# 🚀 Django CRM System

A full-featured Customer Relationship Management (CRM) system built باستخدام Django.

---

## 🔥 Features

* User Authentication (Login / Register / Logout)
* Create, Read, Update, Delete (CRUD)
* Search functionality
* Dashboard interface
* Django Messages (notifications)
* Clean UI with Bootstrap

---

## 🛠️ Technologies Used

* Python
* Django
* HTML / CSS / Bootstrap
* SQLite
